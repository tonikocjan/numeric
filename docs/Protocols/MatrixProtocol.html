  • subscript(_:_:)

    Access the element at the index (i, j). This is equivalent as subscripting first at index ì an then j. Both i and j must be valid indices otherwise the program will crash!

    Example:

    let m: (some) MatrixProtocol = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
print(m[0, 0]) // 1
print(m[2, 1]) // 8

    Swift

  • subscript(_:)

    Access the row at the given position. Index must be a valid index, i.e., greater than or equal to zero and less than height otherwise the program will crash!

    For instance:

    let m: (some) MatrixProtocol = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
print(m[0]) // [1, 2, 3]
print(m[2]) // [7, 8, 9]
