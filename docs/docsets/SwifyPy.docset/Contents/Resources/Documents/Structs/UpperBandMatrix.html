Upper Band Matrix

In mathematics, an upper band matrix is a sparse matrix whose non-zero entries are confined to a diagonal band, comprising the main diagonal and zero or more diagonals above.

It is a square matrix.

  • init(arrayLiteral:)

    Initialize a new Upper Band matrix from the given elements.

    It is expected that provided vectors represent each non-zero diagonal.

    It is required that each vector in elements must be one element shorter than it’s predecesor.

    The bandwithd of the new matrix is equal to elements.count. The size of the matrix is the size of the first vector in elements.

    Example:

    let matrix: UpperBandMatrix = [[1, 2, 3]]
print(matrix)
[[1, 0, 0],
 [0, 2, 0],
 [0, 0, 3]]

    Example:

    let matrix: UpperBandMatrix = [[1, 3, 5], [2, 4]]
print(matrix)
[[1, 2, 0],
 [0, 3, 4]
 [0, 0, 5]]]

    Example:

    let matrix: UpperBandMatrix = [[1, 3, 5, 7], [2, 4, 6]]
print(matrix)
[[1, 2, 0, 0],
 [0, 3, 4, 0],
 [0, 0, 5, 6],
 [0, 0, 0, 7]]
