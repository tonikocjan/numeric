Transposable

The transpose of a matrix is an operator which flips a matrix over its diagonal, that is it switches the row and column indices of the matrix by producing another matrix.

Even though every matrix can be transposed, we provide a separate protocol. Our reasoning is that some data structures, like sparse matrices (BandMatrix, …), cannot provide a transposed version of themselves. Therefore, only some matrices can conform to this protocol.

We might consider implementing this differently, by instead allowing transposed to be any MatrixProtocol:

public protocol Transposable {
  associatedType M: MatrixProtocol
  /// Compute the transpose of this matrix.
  var transposed: M { get }
}
