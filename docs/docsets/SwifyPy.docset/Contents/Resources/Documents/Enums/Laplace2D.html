Laplace2D

public enum Laplace2D

Solving the minimal surface problem using Laplace differential equations.

  • solveBoundaryProblem(fs:fd:fz:fl:h:bounds:)

    Compute the approximate solution to the boundary problem for the Laplace equation in 2D inside the rectangle bounded by (a, b) x (c, d), where the values on the edges are computed with

    u(x, c) = fs(x)
u(b, y) = fd(y)
u(x, d) = fz(x)
u(a, y) = fl(y)
